# Mini shop

## Описание
Mini shop — это веб-приложение для онлайн-продажи товаров с полным циклом: каталог, корзина, оформление заказа, управление заказами через административную панель.

## Функциональность

### Пользовательская часть
- Просмотр каталога товаров
- Фильтрация товаров по категории
- Просмотр карточки товара
- Добавление товаров в корзину
- Оформление заказа
- Личный кабинет пользователя

### Административная часть
- Управление товарами
- Управление категориями
- Управление заказами
- Управление пользователями
